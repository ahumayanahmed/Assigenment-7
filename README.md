#Name: KeenKeeper — Keep Your Friendships Alive

## 📌 Project Overview

**KeenKeeper** is a modern web application designed to help users maintain and strengthen their friendships. It allows users to track communication with friends, view interaction timelines, and stay connected by reminding them when it's time to reach out again.

The goal of the project is to prevent friendships from fading by organizing and visualizing contact history in a simple and engaging way.

---

## 🚀 Technologies Used

* React.js
* Next.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* Vercel (Deployment)

---

## ✨ Key Features

### 1️⃣ Friend Contact Tracking

Users can track how long it has been since they last contacted a friend and easily identify overdue connections.

### 2️⃣ Timeline of Interactions

A visual timeline shows previous interactions such as calls, texts, or meetups, helping users remember their communication history.

### 3️⃣ Smart Status System

Friends are categorized with statuses (e.g., **active**, **due soon**, **overdue**) so users instantly know who they should reconnect with.
